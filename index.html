Python Notepad

A Modern Text Editor Built with Tkinter

Overview

The Python Notepad is a professional, modern text editor application developed using the Tkinter library. It aims to provide a clean, dark-themed interface with essential features expected from a contemporary editor, such as line numbers and a dynamic status bar.

The entire application is encapsulated within the `TextEditor` class, which handles the GUI setup, core text editing functionality, and file operations.

Key Features

  • Dark Theme: Utilizes a professional, eye-friendly color scheme (`#282c34` background, `#abb2bf` foreground).
  • Line Numbers: A dedicated widget on the left side displays the line numbers, synchronized with the main text area scroll.
  • ScrolledText Area: The main text editor uses `tkinter.scrolledtext.ScrolledText` for built-in scrollbar support.
  • Status Bar: A dynamic status bar at the bottom displays the current cursor position (Line, Column) and an overall "Ready" status.
  • File Management: Full support for New (`Ctrl+N`), Open (`Ctrl+O`), Save (`Ctrl+S`), and Save As (`Ctrl+Shift+S`) operations.
  • Unsaved Changes Prompt: Prompts the user to save changes upon closing or opening a new file if the content is modified.
  • Undo/Redo: Built-in support for undo (`Ctrl+Z`) and redo (`Ctrl+Y`) functionality.

How to Run

This application is written in Python and uses the standard library `tkinter`, so no external installations are necessary beyond a Python environment.

Save the code as a Python file (e.g., `main_editor.py`) and run it from your terminal:

python main_editor.py

Color Scheme Details

The application uses a consistent set of dark mode colors:

  • Editor Background: `#282c34`
  • Editor Foreground (Text): `#abb2bf`
  • Line Number Background: `#3e4451`
  • Line Number Foreground: `#61afef` (used for the numbers)
  • Status Bar Background: `#3e4451`
